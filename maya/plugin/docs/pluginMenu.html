Naiad-Buddy for Maya: MayaPlugin Menu

CacheSelectedToEMP

Will bring up a window that allows you to set a path and a framerange. When the user starts the caching it will take all the selected objects in the scene, connect fitting conversion nodes to it and then gather all the bodies into an empSaver. It will then go though the framerange triggering the empSaver

Load Mesh(es) From Emp

Will use the naiadEMPInfo Command to get informations about type and name of the bodies. The window that pops up will use that info to generate a list of what can be loaded from the file. Once the selection is done the user can tell it to import them and a DG network will be setup and the maya meshes will be there. If the UseFrameRange option is specified, then it will set the found framerange onto the EmpLoader, and it will then clamp all frame values inside that range when loading files

CreateEmpLoader

Brings up a file UI and creates a EmpLoader with the given path

CreateNaiadBodyField

Creates a naiad body field, and if particles are selected the field will be attached

CreateNaiadBodyDisplay

Creates a locator node capable of displaying Naiad Bodies(sofar only meshes are supported)
