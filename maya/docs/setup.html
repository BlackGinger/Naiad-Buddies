The easiest way to enable all features of Naiad Buddy For Maya is to source the config files.

  • For Bash : source $NAIAD_PATH/config/naiadForMayaConfig_bash
  • For Csh/Tcsh : source $NAIAD_PATH/config/naiadForMayaConfig_csh
    • Though in more detail the requirements are these: pdcToEmp application:
  • add the $NAIAD_PATH/lib to the LD_LIBRARY_PATH enviroment variable
    • Maya Plugin:
  • add the $NAIAD_PATH/lib to the LD_LIBRARY_PATH enviroment variable
  • add the $NAIAD_PATH/maya/config path to the MAYA_MODULE_PATH enviroment variable

    • Maya then uses the $NAIAD_PATH/maya/config/NaiadBuddyForMayaModule.txt file to determine the location of the scripts, icons and plugin

    To then load the plugin inside maya do one of the following

  • Load it via script editor(MEL) by typing: loadPlugin("naiadForMaya.so")
  • MayaMainMenu->window->Settings/Preferences->Plugin Loader (plugin should be listed there)

    • If loaded succesfully you should see the NaiadForMaya menu in the main Maya Menu on the right hand side.
